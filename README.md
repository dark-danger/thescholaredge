# The Scholar Edge — EdTech & Research Consultancy Platform

The Scholar Edge is a modern, responsive web application designed for an educational research consultancy platform serving students, PhD/M.Phil. scholars, and academicians.

## 🚀 Key Features

- **Thesis & Dissertation Support:** Synopsis/proposal guidance, literature review (PRISMA), methodology, and journal adaptation.
- **Data Analytics Training:** Courses & workshops in IBM SPSS, SmartPLS 4 (PLS-SEM), R & RStudio, Python, Power BI, and Tableau.
- **Research Consulting:** 1-on-1 mentoring by senior statisticians with 25+ years experience.
- **Direct Connect Integrations:** Instant WhatsApp click-to-chat (`+91-86850 99555`) and active Discord community integration.
- **Work Samples & Portfolio Viewer:** Case study previews showcasing statistical outputs and methodological models.

## 💻 Tech Stack

- **Framework:** React + Vite
- **Icons:** Lucide React
- **Styling:** Vanilla CSS (Custom Tokens, CSS Variables, Glassmorphism, Responsive Grid System)

## 🛠️ Getting Started Locally

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Production build
npm run build
```
